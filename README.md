# 📞 Scam Call Detection using NLP

## 🔍 Overview
Real-time end-to-end scam call detection system using
Content Analysis with Dynamic Sparsity Top-K Attention
Regularization — MCA Final Year Project.

## 📊 Results
| Metric | Value |
|--------|-------|
| Classification Accuracy | 90%+ |
| Feature Noise Reduction | 40% |
| False Positive Reduction | 25% |

## 🛠️ Technologies Used
- Java (Core application)
- Python (ML models)
- Natural Language Processing (NLP)
- Attention Mechanisms
- scikit-learn, NumPy, Pandas

## 📌 Key Features
- Real-time scam call detection
- Dynamic sparsity-based attention mechanism
- Signal processing & feature extraction
- Automated alert generation
